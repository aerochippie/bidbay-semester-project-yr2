## Getting Started

### Installing


1. Clone the repo:

```bash
gh repo clone aerochippie/pocket-ecom
```

2. Install the dependencies:

```
npm install
```

3. Build the app:
```
npm run build
```

To run and preview the app, run the following commands:

```bash
npm run preview
```
